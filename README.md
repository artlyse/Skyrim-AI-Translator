# Skyrim AI Translator

Herramienta para traducir mods de Skyrim exportados desde xEdit utilizando diccionarios personalizados e IA local mediante Ollama.

## Características

* Apertura de archivos TXT exportados desde xEdit
* Soporte para múltiples codificaciones e idiomas
* Edición manual de traducciones
* Búsqueda y filtrado
* Estadísticas de progreso
* Guardado y carga de proyectos JSON
* Exportación para xEdit
* Diccionarios personalizados
* Traducción mediante IA local
* Integración con Ollama
* Tema oscuro

## IA soportadas

### Mock

Modo de pruebas sin necesidad de instalar modelos.

### Ollama

Modelos recomendados:

* qwen2.5:1.5b
* qwen2.5:3b
* gemma2:2b
* mistral

## Instalación

pip install -r requirements.txt

## Ejecutar

python main.py
